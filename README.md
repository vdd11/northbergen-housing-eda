# North Bergen, NJ Housing Price Analysis

## Overview
This project analyzes residential property sales in North Bergen, New Jersey
to identify factors that influence housing prices.

## Questions Explored
- What is the distribution of housing prices?
- How do prices vary by property type?
- Does living area impact sale price?
- Are newer homes more expensive?
- Are there seasonal sales trends?

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Findings
- Single-family homes have higher median prices than condos.
- Property size shows a positive relationship with sale price.
- Seasonal trends suggest increased activity in warmer months.
